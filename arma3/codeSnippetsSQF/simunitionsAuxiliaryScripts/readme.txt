folder contains: additional scripts for usage in MP PvE simunition training scenarios

> surrenderOnUncon.sqf - code that makes an AI unit surrender when knocked unconcious. Should be run on the server (recommended to be run in the initServer.sqf file)
> addHitReactionSound.sqf - code snippet for playing a sound when a play is hit by a simunition round
> sounds - folder containing the sound to be played by the addHitReactionSound.sqf code snippet.
